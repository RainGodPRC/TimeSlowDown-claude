隐私政策

TimeSlowDown · 回访分支 · 最近更新：2026-07-08

TimeSlowDown（以下简称"TSD"）的回访分支是一款帮你"回到过去某个瞬间"的记忆 app。我们对你的记忆数据采取最严格的保护：本机优先，不上传。

1. 我们收集什么

我们不收集你的任何个人数据到我们的服务器。具体而言：

  • 你记录的瞬间文本、选择的影像、回访时追加的"现在再看"感受——全部仅存储在你的设备本机（浏览器/应用的 IndexedDB 本地数据库）。
  • 没有账户系统、没有登录、没有手机号或邮箱。
  • 没有第三方统计分析 SDK、没有广告 SDK、没有任何跨应用或跨网站追踪。

2. 数据存在哪里

  • 全部记忆数据存在你的设备本机。
  • 本 app 不向任何服务器发送你的瞬间内容或影像。
  • 导出/分享是你主动的操作——只有你点击"导出记忆包"或"复制分享文案"时，数据才会离开 app，且去向由你决定。

3. 你的权利

  • 导出：随时在"设置 → 记忆保险箱 → 导出记忆包"导出完整、带完整性校验的 JSON 记忆包。
  • 删除：随时清空本机数据，app 会生成删除回执并保留会话内撤销窗口。
  • 可理解：所有 AI 任务可在"AI 任务账本"查看与撤销；AI 永不改写你过去的原话。

4. 权限说明

  • 照片库 / 相机：仅在你主动选择"选一张照片"作为回访入口时使用，影像仅存本机，不上传。
  • 除此之外，本 app 不请求定位、通讯录、麦克风、健康等任何其他权限。

5. 加密与合规

本 app 不包含自定义加密；仅使用系统标准 HTTPS 传输（适用于 Apple 加密出口合规的豁免类别）。儿童与家庭影像建议在家庭/本地保险箱场景单独管理。

6. 联系

如对本政策有疑问，可在 GitHub 仓库 RainGodPRC/TimeSlowDown-claude 提 issue。

本政策适用于 TSD 回访分支（Claude Code 分支）的所有版本。若未来引入云端同步/E2EE 等可选能力，将在此提前更新并征求同意。
